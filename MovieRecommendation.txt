# Условие задачи: Построение графа рекомендаций фильмов

## Цель
Создать базу данных в Microsoft SQL Server для хранения информации о пользователях, фильмах, жанрах, актёрах и их взаимосвязях. Реализовать графовые запросы для анализа данных и визуализировать граф в Power BI.

## Задачи
1. Создать базу данных `MovieRecommendation` с таблицами узлов (`User`, `Movie`, `Genre`, `Actor`) и рёбер (`Knows`, `Likes`, `BelongsToGenre`, `ActedIn`).
2. Заполнить таблицы данными (10 пользователей, 10 фильмов, 10 жанров, 10 актёров, связи).
3. Создать диаграммы связей в Draw.io (`Knows`, `Likes`, `BelongsToGenre`, `ActedIn`).
4. Реализовать SQL-запросы:
   - Запрос 1: Фильмы, которые понравились Alice.
   - Запрос 2: Жанры, которые нравятся друзьям Bob.
   - Запрос 3: Актёры, снимавшиеся в фильмах, которые нравятся Charlie.
   - Запрос 4: Фильмы жанра Sci-Fi, которые нравятся пользователям.
   - Запрос 5: Друзья Alice, оценившие фильмы с Keanu Reeves.
5. Визуализировать граф в Power BI с использованием Force-Directed Graph.
6. Реализовать запросы SHORTEST_PATH.

## Требования
- Использовать SQL Server для базы данных.
- Сохранить диаграммы в формате Draw.io (`movie_graph.drawio`).
- Сохранить Power BI отчёт как `movie_graph.pbix`.
- Предоставить ссылку на Git-репозиторий с файлами:
  - Условие задачи (txt/docx).
  - SQL-скрипт (`movie_graph.sql`).
  - Файл Draw.io (`movie_graph.drawio`).
  - Power BI отчёт (`movie_graph.pbix`).
  - Папка с изображениями для Power BI (`Images/`).
